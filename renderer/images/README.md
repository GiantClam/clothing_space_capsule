## 下载方式

1. 打开Figma设计稿: https://www.figma.com/design/dxeww4feOkNR8UlCxQNokU/yiyiyi?node-id=1-14
2. 找到 Frame 1
3. 选中对应的图片节点
4. 右键 → Export → PNG (2x)
5. 保存到本目录，文件名按上述命名

## 注意事项

- 导出时请选择 2x 分辨率以保证清晰度
- 文件格式必须为 PNG，保留透明背景
- 文件名必须严格按照上述命名，否则页面无法正确显示
- 图片下载完成后，刷新页面即可看到效果
