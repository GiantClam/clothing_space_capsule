# 衣等舱 - AI试衣体验

## 项目简介
衣等舱是一个基于 Electron 的 AI 试衣应用，提供智能服装搭配和虚拟试衣体验。

## 功能特性
- 🎯 智能服装分类选择
- 📸 实时拍照试衣
- 👗 多品类服装支持（上衣、下衣、裙子）
- 🎨 时尚轻奢 UI 设计
- ⚙️ 可配置的系统设置

## 技术架构
- **前端**: HTML5 + CSS3 + JavaScript (ES6)
- **桌面端**: Electron
- **后端**: Node.js + Express
- **图片处理**: 原生 Canvas API

## 开发说明
- 开发模式支持 DevTools (F12 或 Ctrl+Shift+I)
- 支持热重载和实时调试
- 配置页面支持环境变量管理

## 构建说明
```bash
# 安装依赖
npm install

# 开发模式运行
npm start

# 构建应用
npm run build
```

## 注意事项
- 确保 `public/` 目录下的图片文件完整
- 图片格式支持 JPG、PNG、WebP
- 建议图片尺寸保持一致以获得最佳显示效果
